# QuickShop E-Commerce Platform

A sleek, responsive, and fully-featured single-vendor E-Commerce web application built using Django.

## Features
* **Authentication**: Seamless user registration, login, and profile management.
* **Shopping Cart & Wishlist**: Add products to cart, save for later, and manage quantities.
* **Order Management**: Checkout process, payment integration screens, and user order history.
* **Products Catalog**: Detailed product browsing and beautiful UI components.
* **Seller Dashboard**: A dedicated interface for sellers and admins to manage inventory.

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Elijah19Github/quickshop.git
   cd quickshop
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On Mac/Linux:
   source venv/bin/activate
   ```

3. **Install Dependencies:**
   *(Ensure you have Django installed)*
   ```bash
   pip install django
   ```

4. **Run the local Development Server:**
   ```bash
   python manage.py runserver
   ```
   Open `http://127.0.0.1:8000/` in your browser to view the app!
